## 詳細シナリオテスト計画

### 具体的UIテストシナリオ

#### シナリオ1: 新規プロジェクト作成とシンプルモデル構築
**テスト手順**:
1. **起動確認**: `http://localhost:3001` にアクセス → ダッシュボード表示を確認
2. **プロジェクト作成**: 「新規プロジェクト」ボタンクリック → モーダル表示確認
3. **入力テスト**: 
   - プロジェクト名: "テストプロジェクト1" 入力
   - 説明: "UIテスト用プロジェクト" 入力
   - 「作成」ボタンクリック
4. **画面遷移**: プロジェクトエディター画面への自動遷移確認
5. **コンポーネント配置**:
   - 左側パレットの「パイプ」カテゴリクリック → パネル展開確認
   - 「1Dパイプ」クリック → キャンバス中央に配置確認
   - 配置されたパイプの選択状態確認（青枠表示）
6. **境界条件追加**:
   - 「境界条件」カテゴリクリック
   - 「開放端（大気）」クリック → パイプ左端近くに配置
   - 「閉端」クリック → パイプ右端近くに配置
7. **接続作成**:
   - 開放端をパイプ左端にドラッグ → 接続線表示確認
   - 閉端をパイプ右端にドラッグ → 接続線表示確認
8. **プロパティ設定**:
   - パイプ選択 → 右側プロパティパネル表示確認
   - 長さを「2.0」に変更 → 即座反映確認
   - 直径を「0.08」に変更 → 即座反映確認
9. **保存テスト**:
   - Ctrl+S押下 → 保存通知ポップアップ表示確認
   - 3秒後の自動消去確認

**期待結果**: 全ステップが正常完了、エラーなし
**失敗時の記録**: スクリーンショット、コンソールエラー、操作ステップ

#### シナリオ2: E46 M3 VANOSモデル構築テスト
**テスト手順**:
1. **エンジン基盤**:
   - エンジンブロック配置 → プロパティで「6気筒、3.2L」設定
   - 6個のシリンダー配置 → 各シリンダーのVANOS対応バルブ設定
2. **制御系構築**:
   - センサー配置（カムポジション、クランクポジション）
   - 1Dテーブル配置 → VANOSマップデータ設定
   - コントローラー配置 → VANOS制御ロジック設定
   - PIDコントローラー配置 → ゲイン設定
3. **接続テスト**:
   - センサー → コントローラー接続
   - コントローラー → PIDコントローラー接続
   - PIDコントローラー → 制御バルブ接続
4. **パラメータ設定**:
   - VANOSタイミング範囲設定（吸気: 0-40°、排気: 0-20°）
   - PIDゲイン設定（P=1.0, I=0.1, D=0.05）
   - センサー分解能設定

**期待結果**: VANOS制御システムの完全なモデル構築
**検証項目**: 制御ループの完全性、パラメータ妥当性
